# quiz_game

A  Flutter project Quiz Flow with Gamification.
# News App

## Features

- User can play a Quiz game
- Display score
- Score Board also integrated.

### Installation
1. Clone the repository:

2. Install dependencies:
 - flutter pub get
3. Run the app:
 - flutter run
